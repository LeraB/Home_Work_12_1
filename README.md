# Simple slider

## Usage

##### 1. Add markup
```<div id="slideshow-container">

       <div class="mySlides fade">
           <img src="..." style="width:100%">
       </div>

       <div class="mySlides fade">
           <img src="..." style="width:100%">
       </div>

       <div class="mySlides fade">
           <img src="..." style="width:100%">
       </div>

    </div>
```
